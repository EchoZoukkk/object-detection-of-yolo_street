# object-detection-of-yolo_street
This is a team practical project whose core task is to implement object detection for street scenes based on YOLOv8 while achieving maximum model lightweight optimization. We adopt YOLOv8n as the baseline model and conduct a series of optimization strategies including knowledge distillation, model pruning and INT8 quantization.
